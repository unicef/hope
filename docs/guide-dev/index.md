# Developer Guide

Welcome to the HOPE Developer Guide! This documentation will help you set up your development environment and start contributing to the project.

## Quick Start

1. [Setup](setup.md) - Install prerequisites and configure your environment
2. [Running](running.md) - Start the application (multiple options available)
3. [Testing](testing.md) - Run unit and E2E tests
4. [Linting](linting.md) - Code quality and formatting
5. [Stack](stack.md) - Technology overview
6. [Contributing](contributing.md) - Guidelines for contributors

## Development Options

HOPE supports three development workflows:

| Option | Best For | Frontend | Backend |
|--------|----------|----------|---------|
| **Local + Build Watch** (Recommended) | Full integration | Built to Django static | Local venv |
| **Local + Vite Dev** | Frontend dev with HMR | Vite dev server | Local venv |
| **Full Docker** | Production-like env | Vite dev server | Docker container |

Choose the option that best fits your workflow. See [Running](running.md) for detailed instructions.
